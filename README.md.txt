**Developer:**  Abuzar Saleem  
**Internship:** Rhombix Technologies  
**Domain:** Python Development  

## Project Description
This project is a voice-controlled assistant created using Python.  
It listens to user commands and performs tasks like:
- Telling time and date  
- Opening websites (YouTube, Google, Facebook, etc.)  
- Opening Windows apps (Notepad, Calculator)  
- Searching Wikipedia  
- Telling random jokes  
- Introducing itself  

## Technologies Used
- Python  
- `speech_recognition`  
- `pyttsx3`  
- `wikipedia`  
- `webbrowser`  
- `os`, `subprocess`
